Explain me concept of DOM in javascript.
The Document Object Model (DOM) is a tree-like structure that represents your HTML document in memory, created by the browser when your page loads, where each HTML element (like divs, images, inputs) becomes a JavaScript object you can access and manipulate dynamically. For your chat project, use document.getElementById() or document.querySelector() to grab elements like your chat area or search input, then modify them with properties like innerHTML to add messages, textContent to change text, or appendChild() to insert new elements - for example, chatArea.appendChild(newMessage) adds a chat bubble; add event listeners like searchInput.addEventListener('keypress', ...) to detect Enter key presses and update the DOM live, making your interface interactive without page reloads
